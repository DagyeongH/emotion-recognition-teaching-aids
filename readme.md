### Face Classification Model📷
- 목적 : Face Classification Model을 통해서, 자폐아동을 대상으로 표정 인식을 도와주는 프로그램입니다.
- Dataset : image_to_csv.ipynb

- 사용 모델 : Face detection , keras CNN model & OpenCV
- 모델 파일 : face_classification_model.ipynb
